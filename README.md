# Secure Finance Platform

A secure financial management platform built with Blockchain integration and DevOps-driven deployment.

## Tech Stack
- Node.js + Express
- MongoDB (Mongoose)
- TypeScript
- JWT Authentication

## Project Phases
1. Core Finance Platform
2. AI Automation
3. Blockchain Integration
4. DevOps Deployment

## Team Collaboration
